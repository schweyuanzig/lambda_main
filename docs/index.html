1. Setup Your File

All addons must be placed in the following directory on your computer:
ProjectLambda/Addons/YourAddon.lua

2. Define the Metadata

Every addon starts with a table containing its identity. Without these 4 fields, the addon will not load:

name: The title shown in the menu.
version: Your script version.
id: A unique name for the system (no spaces).
box: Choose "left" or "right" to decide which column the UI appears in.

3. The Load Function

The Addons.Load function is where you build your UI. It provides four important tools:

groupbox: Used to add UI elements (buttons, toggles, etc.).
Library: Used for system functions like notifications.
Toggles: A list to check the state of your On/Off switches.
Options: A list to get values from sliders, dropdowns, and inputs.

Example:

local Addons = {}

Addons.name = "..."
Addons.version = "..."
Addons.id = "..."
Addons.box = "..."

function Addons.Load(groupbox, Library, Toggles, Options)
    -- YOUR CODE
end

return Addons

5. Key Reminders

Saving: Always save your file as .lua or .luau
Unique IDs: Ensure your Addon.id and the IDs inside AddToggle or AddSlider are unique to avoid conflicts.
Case Sensitivity: Lua is case-sensitive. Toggles is different from toggles.

FULL EXAMPLE:

local Addons = {}

-- 1. MANDATORY METADATA FIELDS
Addons.name    = "Example Addon"      -- Title displayed in the Addons tab
Addons.version = "1.1.0"              -- Current version of the addon
Addons.id      = "ExampleAddon"       -- Unique identifier for the system
Addons.box     = "left"               -- Column position: "left" or "right"

-- 2. LOAD FUNCTION
function Addons.Load(groupbox, Library, Toggles, Options)
    
    groupbox:AddLabel("This is an information label.")

    groupbox:AddToggle("Example_Toggle", {
        Text    = "Enable Feature",
        Default = false,
        Callback = function(Value)
            print("Toggle changed to:", Value)
        end
    })

    groupbox:AddLabel("Accent Color"):AddColorPicker("Example_Color", {
        Default = Color3.fromRGB(124, 106, 247),
        Title   = "Choose Color",
        Callback = function(Value)
            print("Selected Color:", Value)
        end
    })

    groupbox:AddSlider("Example_Slider", {
        Text     = "Walk Speed",
        Default  = 16,
        Min      = 16,
        Max      = 100,
        Rounding = 0,
        Callback = function(Value)
            print("Slider value:", Value)
        end
    })

    groupbox:AddDropdown("Example_Dropdown", {
        Text     = "Mode Selection",
        Values   = { "Aggressive", "Defensive", "Passive" },
        Default  = 1,
        Multi    = false,
        Callback = function(Value)
            print("Selected Mode:", Value)
        end
    })

    groupbox:AddInput("Example_Input", {
        Text        = "Custom Name",
        Default     = "",
        Placeholder = "Type here...",
        Callback    = function(Value)
            print("User entered:", Value)
        end
    })

    groupbox:AddDivider()

    groupbox:AddButton({
        Text = "Test Notification",
        Func = function()
            Library:Notify("Master Addon loaded successfully!")
        end
    })

    task.spawn(function()
        while task.wait(5) do
            if Toggles["Example_Toggle"] and Toggles["Example_Toggle"].Value then
                local currentSlider = Options["Example_Slider"].Value
                print("System is ACTIVE. Current Slider Value:", currentSlider)
            end
        end
    end)
end

-- 3. MANDATORY RETURN
return Addons
